Installation

Creating a new project

First create a new project:

From your ngM root directory type fab startproject:<project name>,<site_type>,<target directory>, for example: fab startproject:newproject,static,


Currently ngM has the below site types available:

  • basic: Static website
  • simple: Simple website with user managment functionality.
  • gallery: Like simple but also adds a gallery with list/grid view and master/detail pages.
