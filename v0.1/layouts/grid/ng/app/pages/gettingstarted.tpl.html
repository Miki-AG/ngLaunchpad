Getting Started

Managing your local project

Starting the server

Now, from your newly created newproject directory, you can type:

  • fab startserverlocal Starts a local instance of a simple web server for developement.
  • fab stopserver Stops the server.

Adding new files

If you add new files or change project properties (in config.py, i.e., CSS themes) you have to rebuild your project:

  • fab buildlocal Compiles and re-creates your local CSS/JS/HTML
