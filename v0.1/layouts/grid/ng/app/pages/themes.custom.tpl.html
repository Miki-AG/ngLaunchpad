Themes

Custom Themes

You can create your custom theme. You can copy and modify the predefined theme Vintage Polaroid.

Create customized palletes

Use the resource below to generate your own customized color palletes:

https://angular-md-color.com/#/

If you need more custom styling (such as layout changes including padding, margins, etc) you will need to either write CSS rules with custom selectors, or build a custom version of the angular-material.css file using SASS and custom variables.
